# Start Building RESTful API with Golang
**Workshop Overview**

In this workshop, you will build a simple RESTful API using Golang and implement basic CRUD operations. By the end of the workshop, you will have a working Book API.

## Daftar Isi


## Pre-requisites
- Your Favourite IDE
- Go (version 1.20+ recommended). [Download here](https://go.dev/dl)
- Postman. [Download here](https://www.postman.com/downloads/)
- Git. [Download here](https://git-scm.com/install/)
